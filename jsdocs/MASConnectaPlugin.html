Class: MASConnectaPlugin

MASConnectaPlugin

The parent Connecta interface to get objects of MAS Messaging and PubSub interfaces.
Source:
  • MASConnectaPlugin.js, line 79
Examples

To get instance of MASPluginMQTTConstants : An interfaces which contains required MQTT connection options for any MQTT broker

var masMQTTConstants = new MASConnectaPlugin.MASMQTTConstants(host, port, false, "");

To get instance of MASPluginMQTTClient: An interface which corresponds to any Public Broker including MAG

var masMQTTClient = new MASConnectaPlugin.MASMQTTClient(clientId, masMQTTConstants);

Access MASConnectaPluginConstants: A static utility containing the plugin related constants and Global callbacks

var MASConnetaConstants = MASConnectaPlugin.MASConnetaConstants;

To get instance of MASConnectaPluginMessage: An interfaces mapped to the native MASMessage structure

var MASConnectaMessage = new MASConnectaPlugin.MASConnectaMessage();

MASConnecta MASPluginUser APIs are static API which can be called directly using the class reference as below

MASConnectaPlugin.MASUser.sendMessageToUser(successHandler, errorHandler, message, userObjectId);

Methods

(static) MASRegisterListener(successHandler, errorHandler)

This funtion must to be called before using MASConnecta messaging APIs. This API register the notification receiver for observing the message intended to this client
Parameters:
Name Type Description
successHandler successCallbackFunction user defined success callback which will be invoked on success scenario.
errorHandler errorCallbackFunction user defined error callback which will be invoked on failure scenario.

Example

Initializing and registering the listener

	MASConnectaPlugin.MASRegisterListener(
		function(success) {
			if (null != success.Payload) {
				var toAlert = {};
				toAlert.sender = success.SenderId;
				var payload = (success.Payload!=null?success.Payload:success.payload);
				toAlert.payload = atob(payload);
				toAlert.topic = success.Topic;
				alert(JSON.stringify(toAlert));
            }
        }, function(error) {
			console.log("Listener registration failed!!!:" + JSON.stringify(error));
        }
	);
